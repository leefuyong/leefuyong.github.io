字典是另一种可变容器模型，且可存储任意类型对象。python字典的主要属性如下：
       1、通过键而不是偏移量进行读取
       2、任意对象的无序集合
       3、可变长度、任意嵌套
       4、属于可变映射类型，不支持切片操作。
       5、可以看作无序对象引用列表。

创建字典的其它方法

第一种：要求事先可以拼出整个字典

{‘name’:’jone’,’age’:42}

第二种：适合于动态建立字典

d={}

d[‘name’] = ‘jone’

d[‘age’] = 42

第三种：要求键都是字符串

dict(name=’jone’,age = 42)

第四种：键和值分别一一对应在keyslist和valslist

dict([(‘name’,’age’),(‘jone’,42)])
