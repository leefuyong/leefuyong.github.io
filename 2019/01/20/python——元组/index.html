元组由简单的对象组成，与列表非常相似，但是不支持原处修改（因为它是不可变序列），通常由圆括号组成，而列表由中括号构成，具有列表的大多数属性，简单介绍如下：
       1、任意对象的有序集合（有序指的是创建元组对象内容的维持顺序）
       2、通过偏移进行存取
       3、属于不可变序列类型
       4、固定长度、异构、任意嵌套
       5、对象引用的数组

创建元组

元组创建很简单，只需要在括号中添加元素，并使用逗号隔开即可。

 tup1 = (‘Google’, ‘Runoob’, 1997, 2000);
       tup2 = (1, 2, 3, 4, 5 );
       tup3 = “a”, “b”, “c”, “d”; # 不需要括号也可以
       type(tup3)
